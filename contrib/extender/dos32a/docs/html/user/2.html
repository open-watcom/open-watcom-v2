2.0 - DOS/32 Advanced - Installation

In order to install DOS/32 Advanced DOS Extender and the programs accompanying it on your computer you need to run the install.exe installation program. After accepting the terms of the License Agreement the installation program will prompt you for a directory into which you would like to install DOS/32 Advanced and thereafter decompress and copy the files into it. Follow the instructions on the screen, and either press the Enter key to let the install program to modify your system files (autoexec.bat) automatically or select manual system update option to manually alter your system configuration later. Please note that if you have WATCOM C/C++ installed in your computer, the install program will modify the wlink.lnk files in .\WATCOM\BINW and .\WATCOM\BINNT directories if you let it to automatically update your system.

You can find more detailed information about modifications made to your system by the install program in section 5.0 - DOS/32 Advanced - Environment Variable, of this manual.
