3.07 - DOS function 2Fh - Get DTA buffer

 

In: AH = 2Fh
Out: ES:EBX = pointer to DTA buffer

Notes:

a) This function returns the pointer that was previously set by DOS function 1Ah, or the pointer to the default buffer that was allocated by DOS/32 Advanced if DOS function 1Ah was never called.
