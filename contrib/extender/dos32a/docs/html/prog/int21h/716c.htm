3.42 - DOS function 716Ch - Win95 LFN Create/Open File

 

In: AX = 716Ch
EBX = access mode and sharing flags
ECX = file attributes
EDX = action
EDI = alias hint
DS:ESI = pointer to ASCIIZ filename
Out:

if successful:
CF clear
EAX = file handle
ECX = action taken:

01h - file opened
02h - file created
03h - file replaced

if failed:
CF set
EAX = DOS error code
