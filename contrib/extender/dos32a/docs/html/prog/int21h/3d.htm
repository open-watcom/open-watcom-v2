3.15 - DOS function 3Dh - Open File

 

In: AH = 3Dh
AL = access mode
DS:EDX = pointer to ASCIIZ file name
Out:

if successful:
CF clear
EAX = file handle

if failed:
CF set
EAX = DOS error code
