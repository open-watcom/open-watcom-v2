3.10 - DOS function 35h - Get Interrupt Vector

 

In: AH = 35h
AL = INT vector
Out: ES:EBX = selector:offset of interrupt handler

Notes:

a) This function returns a protected mode interrupt vector and is equivalent to DPMI function 0204h.
