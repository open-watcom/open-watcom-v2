2.55 - API function 02h - Get Access to Internal Interrupt Buffers

 

In: AL = 02h
Out: BX = writeable data selector
ESI = pointer to interrupt switches
EDI = pointer to 1-st interrupt buffer

Notes:

a) Selector returned in BX is a writeable data selector that should be used to access needed variables/addresses.
