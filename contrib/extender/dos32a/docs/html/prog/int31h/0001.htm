2.02 - DPMI function 0001h - Free Descriptor

Frees a descriptor.

 

In: AX = 0001h
BX = selector for the descriptor to free
Out:

if successful:
CF clear

if failed:
CF set
AX = error code

8022h - invalid selector

Notes:

a) Each descriptor allocated with DPMI function 0000h must be freed individually with this function, even if it was previously allocated as part of a contiguous array of descriptors.

b) DOS/32 Advanced DPMI and DPMI 1.0 hosts will zero any segment registers which contain the selector specified in register BX.
