4.0 - Extended VGA/VBE functions - Overview

One of DOS/32 Advanced most powerful features is that it extends the most popular VESA VBE 2.0 functions. This frees protected mode programmers from the need to use DPMI calls in order to access VESA VBE. DOS/32 Advanced is the first DOS Extender that truly extends VBE 2.0 calls and supports all of currently defined VBE 2.0 functions!

Functions listed below have been extended and are available in protected mode through software interrupt 10h (INT 10h). The function number is placed in the AX register.

Please note that this is only a brief documentation of VESA VBE 2.0 calls that are supported by the DOS Extender. You will probably need the full VBE 2.0 documentation which contains description of tables and returned values if you are planning to program a software that seriously uses VESA VBE 2.0 calls.
