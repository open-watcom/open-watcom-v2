3.2 - SUNSYS Setup - Environment String Window

DOS/32A Environment String Window
The Environment String Window will show the currently defined DOS32A environment variable (can be defined in the autoexec.bat file). Please refer to DOS/32 Advanced User's Reference documentation manual for more information about DOS/32 Advanced environment variable DOS32A.
