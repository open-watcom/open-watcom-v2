2.06 - DOS/32 Advanced D32A Library function - d32a_get_dpmi_version()

Synopsys:

#include <d32a.h>
int d32a_get_dpmi_version(void);


Description:

This function returns the version number of the DPMI host under which the application is currently running (usually 0.9 or 1.0).


Returns:

Version of the currently present DPMI host. For example in a hypothetical DPMI version 3.75 the returned value will be 0x034B, and a common DPMI implementation version 0.9 will be returned as 0x005A.
